Description: endocal is a cross-platform, compact GUI application for the optical distortion calibration of fluid-immersed endoscopes (https://github.com/gift-surg/endocal).
        endocal was developed by Dzhoshkun I. Shakir as part of the GIFT-Surg project (http://www.gift-surg.ac.uk) at the Translational Imaging Group in the Centre for Medical Image Computing at University College London (UCL).
