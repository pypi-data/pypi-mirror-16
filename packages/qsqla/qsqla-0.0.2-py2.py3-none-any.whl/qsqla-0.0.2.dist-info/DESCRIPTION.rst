qsqla
===============================


qSQLA is a query builder for SQLAlchemy Core Selectables 

Installation / Usage
--------------------

To install use pip:

    $ pip install qsqla


Or clone the repo:

    $ git clone https://github.com/hoffmann/qsqla.git
    $ python setup.py install
