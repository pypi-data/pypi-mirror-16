Passiter: A simple password generator
=======================
Usage:

.. code-block:: python

    import passiter
    p = passiter()
    for _ in range(10):
        print(next(p))
    # bA2x9m,P
    # b:vOj29N
    # sN%6gCh4
    # ,khl0M5O
    # jZ3F1v{d
    # H5Atra^1
    # t$7qM4Zv
    # 7lUi6[Tj
    # Y6p5cw_E
    # Db>Qn45p
----
